Справка

После выбора пункта меню «Авторизация» в правом верхнем углу идёт заполнения имя пользователя и пароля как показано на рисунке 1.

Рисунок 1 – Форма авторизации

Если пара пользователь и пароль были введены, верно, то авторизация пройдёт успешно. Успешная авторизация даёт возможность просмотра и редактирования содержимого базы данных. Это даёт возможность использовать программный продукт, так как без успешной авторизации использование невозможно. После авторизации произойдёт переход на главную страницу. Пользователя может создать только администратор через специальную панель администратора, адрес которой недоступен через меню.

После перехода на главную страницу пользователь получает доступ к базе данных для редактирования, удаления или добавления элементов.

Для алгоритма требуются данные объектов, автомобиля, доставляющего заказ, место/места доставки, данные заказы и объекты для погрузки.

Если в базе данных нет ни одного объекта, то требуется добавить объекты. Для этого нужно выбрать подпункт «Добавить объект» в меню.

Для того чтобы добавить объект в базу нужно выбрать пункт меню «Объекты» и соответствующий подпункт добавить объект. Когда появится форма заполнения как показано на рисунке 2, нужно ввести идентификатор объекта, наименование, производителя, вес (в килограммах), партию, цену и номер склада, где находиться объект и нажать кнопку добавить.

Рисунок 2 – Форма добавления объектов

После добавления объекта он появится в списке объектов. Для просмотра списка объектов нужно выбрать пункт меню «Объекты» и подпункт список объектов.

Появится форма со списком объектов. Для редактирования свойств объекта нужно щёлкнуть на выделенный идентификатор объекта, а для удаления объекта щёлкнуть на значок корзины (см. Рис. 3).

Рисунок 3 – форма просмотра и редактирования объектов

При щелчке на изменение свойств объектов появляется форма, где расположены текущие свойства выбранного объекта, и изменяемые свойства объекта как показано на рисунке 4. По умолчанию в изменяемых свойствах расположены текущие значения. Для изменения свойства, нужно стереть текущее значением во вводимом поле и ввести новое. После ввода всех изменяемых значений, нужно нажать кнопку изменить параметры.

Рисунок 4 – Форма изменения свойств объектов

Далее нужно ввести данные машины. Для этого нужно выбрать пункт меню «Автомобили» и подпункт «Добавить машину». Чтобы добавить машину нужно заполнить форму, показанную на рисунке 5 и нажать кнопку добавить.

Рисунок 5 – Форма добавления машины

После создания нужной машины, нужно создать новый заказ. Для этого нужно заполнить форму, вызываемую в меню «Заказы» подпункта «Добавить заказ», показанную на рисунке 6.

Рисунок 6 – Форма добавления заказа

Добавив заказ, нужно заполнить список вещей для заказа. Для этого нужно выбрать подпункт добавления заказа в пункте меню «объекты для погрузки» и заполнить форму, показанную на рисунке 7. После заполнения формы следует нажать кнопку добавить для внесения изменений.

Рисунок 7 – Форма добавления объекта для погрузки

После заполнения списка всех вещей для заказа нужно выбрать пункт меню «Карта расположения». После заполнения идентификатора заказа, нужно выбрать метод оптимизации и нажать кнопку «Составить карту» (см. рис. 8).
